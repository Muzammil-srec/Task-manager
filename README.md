# 🚀 Task Manager Web App

A modern and responsive Task Manager built using HTML, CSS, and JavaScript with full CRUD functionality and local storage support.

---

## ✨ Features
- ➕ Add tasks dynamically  
- ✏️ Inline edit with save option  
- 🗑️ Delete tasks  
- ✔️ Mark tasks as completed  
- 💾 Persistent storage using localStorage  
- 🎨 Clean and modern UI  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript
